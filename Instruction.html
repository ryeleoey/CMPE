CMPE Instruction

Welcome to the CMPE Instruction page.

Step by step instructions to follow.

1. Install Github,Git, and VS Code on your computer.

2. Create a Github account if you don't have one.

3. Connect your Github account to VS Code.

4. Create a new repository on Github.

Simple commands to use Git.

1. git init: Initialize a new Git repository.

2. git add: Stage changes for commit.

3. git commit -m "Your commit message": Commit changes with a message.

4. git push: Push changes to the remote repository.
